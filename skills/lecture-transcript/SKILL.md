# 大学讲课逐字稿生成

## 目标
生成大学老师课堂上实际讲课的逐字稿，而非PPT内容的复制或教科书式的定义描述。

---

## 风格要求

### 讲课场景设定
- **角色**: 大学授课老师
- **场景**: 教室课堂讲授
- **对象**: 本科学生
- **语气**: 自然口语化、亲切引导式

### 语言风格对比

| ❌ 不应有的风格 | ✅ 应有的风格 |
|----------------|--------------|
| PPT定义直接复制 | 口语化解释概念 |
| 教科书式描述 | 课堂讲解式表达 |
| 抽象理论陈述 | 具体例子引入 |
| 机械罗列要点 | 有逻辑的讲解串联 |
| "排序是..."（定义式） | "什么是排序呢？大家想想..."（引导式） |

---

## 写作指南

### 开场引入
```
❌: 排序是计算机程序设计中的一种重要操作。

✅: 同学们好。今天我们开始学习排序。排序大家应该都不陌生——生活中到处都在用排序。比如你打扑克牌，摸牌后整理手里的牌，这就是排序。那具体来说，排序是什么意思呢？
```

### 定义讲解
```
❌: 排序的功能是将一个数据元素（或记录）的任意序列，重新排列成一个按关键字有序的序列。

✅: 排序，简单说就是把乱的变成有序的。给一堆数据，按某个标准排好顺序。比如按成绩高低排学生名单，按价格排商品列表。这个"标准"在数据结构里叫关键字——就是用来比较的那个字段。
```

### 概念展开
```
❌: 排序分为内部排序和外部排序。内部排序是指待排序记录存放在计算机随机存储器（内存）中进行的排序过程。

✅: 排序分两大类：内部排序和外部排序。什么区别？内部排序，数据都在内存里——内存够大，数据全放进内存，直接排序。外部排序呢？数据太多，内存装不下，得借助外存——磁盘、硬盘，分批处理。我们这门课主要讲内部排序，外部排序以后有机会再说。
```

### 算法讲解
```
❌: 直接插入排序的基本思想是将一个记录插入到已排好序的有序表中，从而得到一个新的、记录数增1的有序表。

✅: 先讲第一种方法——直接插入排序。这个思想大家应该能理解。想想你打扑克牌，每摸一张牌，是不是把它插到手里已有牌的合适位置？小的放左边，大的放右边——这就是插入排序。一张一张插，手里的牌始终保持有序。
```

### 代码讲解
```
❌: [直接列出算法代码]

✅: 来看具体怎么实现。假设我们有个数组，从第二个元素开始，每个元素往前找位置插入。为什么从第二个开始？因为第一个元素天然有序——一个数不需要排。第二步，把当前元素和前面的比较，找到合适位置，插入进去。怎么找位置？从后往前比，比到比它小的就停下，插在后面。
```

## 结构要求

### 段落格式
- **章节标题**: 加粗（如 **p1 章节引入**）
- **讲解段落**: 普通文本，每段一个完整讲解单元
- **编号要点**: 加粗（如 1. xxx、第一、①）

### 讲解单元组成
每个讲解单元应包含：
1. **引入话**: "大家想想"、"举个例子"、"来看这个"
2. **核心内容**: 用通俗语言讲解概念/算法
3. **实例说明**: 具体数字、具体场景
4. **过渡衔接**: "接下来"、"那么"、"所以"

---

## 禁止行为

1. **禁止直接复制PPT定义** — 必须转换为口语讲解
2. **禁止机械罗列** — 要有讲解逻辑和衔接
3. **禁止过于书面化** — 避免"其"、"即"、"所谓"等书面用语
4. **禁止缺少互动感** — 要有"同学们"、"大家看"等称呼

---

## 处理流程（防止超时）

当PPT内容较多时，直接一次性生成会超时失败。必须按以下流程处理：

### 1. 先识别章节结构
首先分析PPT内容，识别出所有章节（按PPT的章节划分或内容逻辑划分），列出章节清单。

### 2. 按章节逐个生成
**必须一个章节一个章节地处理，不能一次性处理所有章节！**

流程：
```
第一轮：处理第1章节 → 生成文档（如：逐字稿_第1章.md）
第二轮：处理第2章节 → 生成文档（如：逐字稿_第2章.md）
第三轮：处理第3章节 → 生成文档（如：逐字稿_第3章.md）
...
最后一轮：合并所有章节文档 → 最终文档（如：逐字稿_完整版.md）
```

### 3. 每轮处理规则
- 每轮只处理**一个章节**
- 处理完一个章节后，等待用户确认继续下一章节
- 或使用"继续处理下一章节"指令自动推进
- 每个章节单独保存为一个文档文件

### 4. 最终合并
所有章节完成后，将所有章节文档合并为一个完整文档：
- 按章节顺序拼接
- 保持章节标题格式一致
- 添加总标题和目录（可选）

## 示例

**p12 插入排序基本思想**

同学们好。今天我们学习插入排序。

插入排序的思想其实很好理解，大家在生活中都用过。想想你打扑克牌的时候，每摸一张牌，你都会把它插到手里已有的牌中合适的位置——小的插左边，大的插右边，手里的牌始终保持有序。这就是插入排序！

基本思想是这样的：每一趟将一个待排序的记录，按其关键字大小插入到前面已经排好序的子序列中的适当位置。一直插入，直到全部记录插入完成。

插入排序有几种具体方法：直接插入排序、折半插入排序、希尔排序。我们先讲最基础的直接插入排序。
